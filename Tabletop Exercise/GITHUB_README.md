# Enterprise Incident Response Program

Production-ready IR framework for federal contractors. 4 phases (~35K words), 3 threat playbooks, tabletop validation with 8 discoverable gaps. NIST 800-61 aligned, CUI protection focused.

---

## What's Here

**Phase 1: Framework & Procedures**
- Incident classification, IR team roles, escalation procedures
- Board escalation criteria, decision authority matrix
- Evidence preservation standards (7-year retention)

**Phase 2: Detection & Triage**
- TRIAGE investigation framework (Timeline, Reach, Impact, Attacker, Gather, Escalation)
- Attack path mapping, evidence standards, root cause procedures
- Investigation completion checklist + report template

**Phase 3: Response Playbooks**
- Ransomware (detection, containment, recovery, ransom authority matrix)
- Data Exfiltration (DLP indicators, breach scope, notification procedures)
- Supply Chain (vendor breach, access revocation, remediation verification)

**Phase 4: Post-Incident & Continuous Improvement**
- Forensic analysis procedures, root cause analysis (5-why methodology)
- Lessons learned process, procedure updates, board briefing template
- Continuous monitoring, incident closure criteria, annual trend analysis

**Tabletop Exercise Validation**
- 3 realistic scenarios (Ransomware, Data Exfil, Supply Chain)
- 8 gaps discovered during tabletop: [X] Finding → [Y] Impact → [Z] Remediation
- Procedure updates showing continuous improvement
